# 2026-04-10

## Webinar funnel automation project (new idea)
- Goal: automate deployment of recorded webinars into a full funnel + follow-up system.
- Desired capabilities:
  - Generate a webinar landing page + thank-you/replay pages.
  - Lead capture routed into Lawmatics with consistent tagging.
  - Automated reminders and post-webinar follow-up through Lawmatics.
  - Outbound call follow-up by a live intake specialist (potentially offshore), with structured call queue and dispositions.

## Proposed architecture (high-level)
- Treat each webinar as a deployable “campaign” with standard inputs/outputs.
- Split into two subsystems:
  1) Campaign Generator (pages/assets) — recommended Webflow templates/CMS for speed.
  2) CRM Orchestration — Lawmatics as source of truth for contacts, tags, workflows.
- Glue/automation layer (Phase 2+): Make.com (preferred) or Zapier to sync registrations/attendance and create tasks.

## Suggested implementation phases
- Phase 1 (1–2 webinars): manual page build + Lawmatics-driven reminders/follow-up; manual attendance tagging; manual daily call list.
- Phase 2: automate campaign creation + attendance tagging (webhooks/exports → Lawmatics tags).
- Phase 3: add outbound calling system + QA (scripts, dispositions, escalation, recordings where legal).

## Default stack recommendation (pending answers)
- Webinar platform: Zoom Webinar (or Zoom Meeting initially).
- Landing pages: Webflow.
- Messaging: keep reminders/follow-up in Lawmatics initially to avoid multi-system duplication.
- Outbound calls: firm-owned US VoIP number; avoid using caller’s personal number.

## Open questions to tailor blueprint
- Webinar type: live/evergreen/both.
- Platform choice.
- Lawmatics SMS capability/number.
- Primary CTA.
- Jurisdiction/disqualifiers.
